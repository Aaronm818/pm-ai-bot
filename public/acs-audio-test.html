Instructions

  • Regular Client: Standard WebSocket client for messaging
  • ACS Client: Simulates Azure Communication Services audio streaming
  • Audio Metadata: Sends stream configuration (first message from ACS)
  • Audio Data: Sends PCM audio data packets
  • Stop Audio: Stops audio playback

ACS Message Formats:

Audio Metadata:

{"kind": "AudioMetadata", "audioMetadata": {...}}

Audio Data:
